What This Page is About

I created this page as part of my CSIT121 Lab 02 assignment. It demonstrates the steps a website publisher needs to follow in order to create and publish a functional webpage online.

Things a Website Publisher Needs

  • You need a VS Code editor and a GitHub account to host projects
  • You need to know how HTML basics works using semantic elements and web standards, and an internet browser to test and validate the page

Steps to Publish a Web Page

  1. Create a project folder (for example, lab02).
  2. Write your code in index.html using semantic elements.
  3. Validate the HTML to fix errors and warnings.
  4. Commit and push the folder to your GitHub repository.
  5. Publish the link.

Most of these things I learnt from MDN: Creating the content, MDN: Dealing with files, and MDN: What will your website look like.
